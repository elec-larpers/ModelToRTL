# ModelToRTL
### The point of this repository is to research and implement a way to design an ASIC that essentially 'runs' a convolutional neural network model. 


## We will acheive this using the following technologies 
- YOLO model (TBD)
- Convert this YOLO model (native python) to (c++)
- Convert our c++ code to an RTL 
- Synthesize our RTL to some sort of ASIC 
- Implement this ASIC alongside other components as part of our submission

## Relevant Tools 
- Python to c++: hls4ml
- c++ to RTL: Bambu by pandA lab
- RTL to ASIC: OpenLane
